Should the MVP resource directory include affiliate links?

The directory lists load boards, fuel cards, factoring companies, ELDs, etc. Affiliate links earn a referral commission when users click through and sign up.

A

No affiliate links in MVP

Directory listings are purely editorial — best-for descriptions, neutral links, no tracking parameters. Builds trust faster. Affiliate infrastructure added later once the directory is proven useful.

Best for: Credibility-first approach. Users trust unbiased listings.

B

Affiliate links where available from day one

Add affiliate tracking to directory links where programs exist (DAT, 123Loadboard, ELD providers, fuel cards, etc.). Disclose clearly with an "affiliate" badge. Starts generating revenue immediately.

Best for: Early monetization signal to validate the directory's value.

C

Build affiliate-ready links, don't activate yet

Structure directory links to support affiliate parameters from the start (easy to swap in later), but launch with plain links. No disclosure needed now, no revenue now — but no migration work later either.

Best for: Clean MVP with a clear upgrade path.
